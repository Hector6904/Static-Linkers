# Static Linkers

Static Linkers is a competitive Capture The Flag (CTF) team focused on solving cybersecurity challenges across multiple domains.

This repository serves as a structured archive of our writeups, technical analyses, and methodologies from various competitions.

## Domains Covered

- Digital Forensics
- Cryptography
- OSINT
- Web Exploitation
- Reverse Engineering
- Programming
- Miscellaneous Challenges

## Purpose

The goal of this repository is to document our approach, reasoning, and technical processes while solving CTF challenges.

Each writeup aims to:
- Break down the challenge clearly
- Explain the thought process behind the solution
- Share techniques that may help others learn

All content is provided for educational purposes only.
